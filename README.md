# Smart.House.IoT.Project
This project was made with a NodeMCU ESP8266 WiFi module, Raspberry Pi4, Temp and Humidity sensor, Flame sensor, Photoresistor, RGB LED and a Buzzer.

Buzzer  <-----------------|-------|         |-------------------------|
